# bplist-reader-js

bplist-reader-js is a reader library for Apple Binary Property List (*.plist).

## Support Types

| Abstract Type | JS/TS Type |
| --- | --- |
| Boolean | boolean |
| Integer | number \| bigint |
| Floating-point value | number |
| String | string |
| Data | ArrayBuffer |
| Date | Date |
| Array | unknown[] |
| Dictionary | Record<string, unknown> |
